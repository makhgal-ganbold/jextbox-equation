JExtBOX Equation
==============================

Joomla! Extension

Author [Galaa](https://www.galaa.net/ "Galaa's Personal Page") Publisher [JExtBOX](https://www.jextbox.com/ "JExtBOX - BOX of Joomla Extensions").

**Description**

This plugin helps to display math equations, diagrams and figures on web pages, and it supports equation numbering.
